# SaveTools 命令参考

## 礼包命令

主命令：`/savekits`  
简写：`/sk`

| 命令 | 描述 |
|----------|------|
| `/sk help` | 查看礼包命令帮助 |
| `/sk create <礼包名>` | 创建礼包并打开编辑界面 |
| `/sk edit <礼包名>` | 编辑已有礼包 |
| `/sk look <礼包名>` | 预览礼包内容 |
| `/sk give <礼包名>` | 自己领取礼包 |
| `/sk giveplayer <礼包名> <玩家|*> [数量]` | 给指定玩家或全部在线玩家发放礼包 |
| `/sk delete <礼包名>` | 删除礼包 |
| `/sk list` | 查看礼包列表 |
| `/sk setmaxclaims <礼包名> <次数>` | 设置每个玩家最大领取次数 |
| `/sk setinterval <礼包名> <间隔>` | 设置礼包领取间隔 |

### 礼包领取限制

`/sk setmaxclaims <礼包名> <次数>`：

| 值 | 含义 |
|----------|------|
| `-1` | 不限制领取次数 |
| `1` 或更大整数 | 每个玩家最多可领取的次数 |

`0` 是无效值。

`/sk setinterval <礼包名> <间隔>` 支持：

| 写法 | 含义 |
|----------|------|
| `60` | 60 秒 |
| `30s` | 30 秒 |
| `5m` | 5 分钟 |
| `1h30m` | 1 小时 30 分钟 |
| `1d` | 1 天 |
| `-1` | 不限制间隔 |

修改领取间隔后，插件会清空该礼包已有的上次领取时间记录。

### `/sk list` 行为

有 `savekits.edit` 权限的玩家执行 `/sk list` 会打开礼包列表 GUI。  
没有该权限的玩家会收到文本形式的礼包列表。

在礼包列表 GUI 中：

| 操作 | 结果 |
|----------|------|
| 左键礼包 | 预览礼包 |
| 右键礼包 | 编辑礼包，需要 `savekits.edit` |
| 上一页/下一页 | 切换页面 |

## 保存物品命令

主命令：`/saveitem`  
简写：`/si`

| 命令 | 描述 |
|----------|------|
| `/si help` | 查看保存物品命令帮助 |
| `/si save <名称>` | 保存手持物品 |
| `/si give <名称>` | 自己领取已保存物品 |
| `/si giveplayer <名称> <玩家> [份数]` | 给指定在线玩家发放物品 |
| `/si delete <名称>` | 删除已保存物品 |
| `/si list` | 查看已保存物品列表 |

`/si giveplayer` 的目标玩家必须在线。`[份数]` 不填时默认为 `1`，填写时必须是大于 `0` 的整数。

## 插件命令

主命令：`/savetools`  
简写：`/st`

| 命令 | 描述 |
|----------|------|
| `/st help` | 查看 SaveTools 帮助 |
| `/st reload` | 重载配置、语言、GUI 和缓存数据 |

## 使用示例

```bash
/si save 活动钥匙
/si giveplayer 活动钥匙 Steve 3

/sk create 每日礼包
/sk setmaxclaims 每日礼包 -1
/sk setinterval 每日礼包 1d
/sk giveplayer 每日礼包 * 1
```
