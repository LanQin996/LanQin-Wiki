# 关于 SaveTools

SaveTools 是一个用于保存物品和管理礼包的 Minecraft 服务器工具插件。它适合用来做活动奖励、新手礼包、商城物品、补偿发放等功能。

## 插件信息

| 项目 | 内容 |
|----------|------|
| 作者 | `LanQin_` |
| 当前版本 | `2.1.2` |
| Java 版本 | `Java 8+` |
| 构建目标 | `Spigot API 1.12.2` |
| 兼容 | `Spigot`、`Paper`、`Folia` |
| 可选前置 | `PlaceholderAPI` |

## 功能概览

### 保存物品

- 保存手持物品到数据文件或数据库
- 按名称发放保存物品
- 支持给指定在线玩家发放多份
- 支持 PlaceholderAPI 显示保存物品描述

### 礼包

- 创建、编辑、预览、删除礼包
- 关闭编辑 GUI 自动保存
- 支持最大领取次数
- 支持领取间隔
- 支持给单个玩家或全部在线玩家发放
- 支持礼包冷却 PAPI 变量

### 数据存储

- 默认 YAML 存储
- 可选 MySQL 存储
- MySQL 模式可配合自动刷新缓存用于多服共享数据

## 适用场景

- 新手礼包
- 每日礼包、周礼包
- 活动奖励
- 商城物品发放
- 管理员快速保存和复用复杂物品
