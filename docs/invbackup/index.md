### 第一步：安装InvBackup插件
1. 下载 `InvBackup.jar` 文件
2. 放入 `plugins` 文件夹
3. 重启服务器
4. 检查控制台是否有错误信息

### 第二步：基础配置
1. 编辑 `plugins/InvBackup/config.yml`
2. 根据需要调整基础设置
3. 输入 `/invbackup relaod` 重载插件
