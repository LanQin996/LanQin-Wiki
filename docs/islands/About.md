# 关于 Islands


**Islands** 是一个为 Minecraft 服务器提供**团队独立岛屿世界**玩法的插件，每个玩家都拥有完全独立的岛屿世界，支持多模板系统、团队管理、边界控制、升级与经济集成、以及丰富的 PlaceholderAPI 变量。与传统空岛插件不同，Islands 为每个玩家创建专属的岛屿世界，提供更私密和个性化的生存体验。

## 插件信息

- 作者: `LanQin_`
- 版本: `1.0.11`
- 支持版本: `1.12.2+`

## 特性概览
#### 空岛系统
- **空岛**: 每个玩家可以创建的个人世界
- **团队**: 空岛可以邀请其他玩家组成团队
- **等级**: 空岛可以升级，扩展边界和功能

#### 团队管理
- **岛主**: 创建空岛的玩家，拥有最高权限
- **成员**: 被邀请加入的玩家
- **访客**: 进入空岛但非团队成员的玩家

#### 权限控制
- **锁定**: 只有团队成员可以进入
- **PVP**: 控制是否允许战斗
- **物品控制**: 控制拾取和丢弃物品
