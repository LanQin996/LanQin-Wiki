# 添加收款通道

## 1. 添加收款通道

### 1.1 二维码模式

::: tip 操作步骤
1. 点击 **添加**  
2. 填写通道名称  
3. 选择支付方式  
4. 选择通道  
5. 点击 **下一步**  
:::

![二维码模式-步骤一](https://cdn.jsdu.cn/opdav/20250915/05018d0bb4cd18b0fe6d7c133e80f523.png)

---

::: tip 操作步骤
1. 选择 **网关**  
2. **二维码模式** 可根据喜好自由选择  
3. **订单检查方式** 与 **取消订单递增金额** 建议保持默认  
4. 点击 **保存** 完成设置  
:::

![二维码模式-步骤二](https://cdn.jsdu.cn/opdav/20250915/494b1b5bd84a94828f7bdb9963883360.png)

---
