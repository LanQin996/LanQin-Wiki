# 添加收款通道与监控软件

## 1. 添加收款通道

### 1.1 二维码模式

::: tip 操作步骤
1. 点击 **添加**  
2. 填写通道名称  
3. 选择支付方式  
4. 选择通道  
5. 点击 **下一步**  
:::

![二维码模式-步骤一](https://cdn.jsdu.cn/opdav/20250915/2dd57101456a41dfc11f255997275c0d.png)

::: tip 操作步骤
1. 点击 **上传并解析二维码**  
2. 选择收款码类型  
3. 点击 **确定添加**  
:::

![二维码模式-步骤二](https://cdn.jsdu.cn/opdav/20250915/95a134a469aee8eedbf89eee54c76be8.png)

---

### 1.2 使用支付宝 PID 模式

::: tip 操作步骤
1. 点击 **添加**  
2. 填写通道名称  
3. 选择支付方式  
4. 选择通道  
5. 收款码类型选择 **PID**  
6. 填写支付宝 PID  
7. 二维码模式选择 **模式 9**  
8. 点击 **保存**  
:::

::: details 手机支付宝扫码获取 PID
![获取PID-步骤一](https://cdn.jsdu.cn/opdav/20250915/0731125c94435547ab2349586c52ce1b.png)  
:::
![获取PID-步骤二](https://cdn.jsdu.cn/opdav/20250915/d47943c8dccddb36f7aae25fffc159d4.png)
