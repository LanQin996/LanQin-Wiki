# CooPay 
**是一款专业的 Minecraft 收款插件，为各大服主提供一系列的简洁快捷方便的收款方式**

自2023年8月开始至今，经历多次迭代更新，目前功能均已完善。

## 🚀 核心功能
+ **自定义金额支付** - 支持玩家自定义支付金额
+ **灵活奖励系统** - 支持控制台和OP身份执行奖励命令
+ **一键补偿功能** - 支持服务器换周目时一键补偿玩家
+ **实时订单查询** - 支持查询订单状态和支付信息
+ **自定义礼包功能** - 支持自定义多个礼包奖励以及价格

## 🎯 支付方式

- **微信支付** (wx) - 支持扫码支付
- **支付宝** (zfb) - 支持扫码支付
