# CooPay 插件命令参考

## 📋 命令总览

### 主命令
```
/cp [子命令]
/coopay [子命令]
/coo [子命令]
```

### 命令分类
- 💰 **支付相关**: 发起支付、查看余额
- 🎁 **奖励管理**: 奖励设置、补偿处理
- 📊 **信息查询**: 统计查看、订单查询
- ⚙️ **系统管理**: 重载、调试、配置

## 💰 支付相关命令

### 发起支付
```
/cp trade <支付方式> <金额>
```

**功能**: 发起支付订单

**参数**:
- `支付方式` (必需): wxpay(微信) 或 alipay(支付宝)
- `金额` (必需): 支付金额，支持小数

**权限**: 无

**示例**:
```bash
/cp trade wxpay 100    # 使用微信支付100元
/cp trade alipay 50.5  # 使用支付宝支付50.5元
/cp trade wxpay 1000   # 使用微信支付1000元
```

**成功消息**: 
- `&a创建订单成功！`
- `&a订单支付二维码生成成功！`
- `&a正在创建一个新的订单 订单号: {trade}`

**失败情况**:
- 指令格式错误: `&c指令格式错误 正确使用方式/cp trade 支付方式 金额`
- 非法数字: `&c输入了非法的数字！`
- 非法支付方式: `&c不存在的支付方式`
- 订单创建失败: `&c订单由于未知原因创建失败！请联系服务器管理员！原因:{cause}`
