# CooPay 插件命令参考


## 命令总览

### 主命令
```
/cp [子命令]
/coopay [子命令]
```

## 支付相关命令

### 发起支付
```
/cp trade <支付方式> <金额>
```

**功能**: 发起支付订单

**参数**:
- `支付方式` (必需): wx(微信) 或 zfb(支付宝)
- `金额` (必需): 支付金额

**权限**: 无

### 购买礼包
```
/cp buy <礼包> <支付方式>
```

**功能**: 购买指定礼包

**参数**:
- `礼包` (必需): 礼包名称
- `支付方式` (必需): wx(微信) 或 zfb(支付宝)
