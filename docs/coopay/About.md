# CooPay
<update />

::: tip 提示
**是一款专业的 Minecraft 收款插件，为各大服主提供一系列的简洁快捷方便的收款方式**<br>
自2023年8月开始至今，经历多次迭代更新，目前功能均已完善。
:::


## 核心功能

+ **多平台支付支持** - 支持 CPAY 和 XPAY 两个支付平台
+ **混合平台模式** - 支持不同支付方式使用不同平台（如微信用CPAY，支付宝用XPAY）
+ **自定义金额支付** - 支持玩家自定义支付金额
+ **灵活奖励系统** - 支持控制台和OP身份执行奖励命令
+ **一键补偿功能** - 支持服务器换周目时一键补偿玩家
+ **实时订单查询** - 支持查询订单状态和支付信息

## 支付方式

- **微信支付** (wxpay) - 支持扫码支付
- **支付宝** (alipay) - 支持扫码支付

## 平台配置

### 单一平台模式
所有支付方式使用同一个平台

### 混合平台模式
- 微信支付 → CPAY 平台
- 支付宝 → XPAY 平台
